Lesson 02 - Understanding variables and basic math operations
Hi, in this lesson we'll understand what variables are and how they are important for coding. For this lesson we'll create some variables and use it with some other commands and also apply some math functions.

We'll also learn how to use the four basic math operations:
  • Addition
  • Subtraction
  • Multiplication
  • Division
Using variables in a program
We've learnt that all programs begin with begin and end with an end and that inside that block we can create our program by entering instructions or commands. In order to use variables, we need to tell the code that we will use them by entering the word variables and naming the variables that we are going to use.

One important note is that variable names should not have spaces or special characters. When we have to use more than one word for a variable, we can use the underline symbol or separate it by using lower and upper case. For example, if we have two variables that we want to call "first operand" and "second operand", in coding we should use something lile firstOperand and secondOperand, or first_operand and second_operand.

Below is an example on how we use variables, and how we print it to the screen using the println command: 
variables
  firstName, lastName
begin
  firstName = "Abraham"
  lastName = "Lincoln"
  println("The name is ", firstName, " ", lastName)
end
The program above will display the text "The name is Abraham Lincoln".

Another important concept is that variables can receive values from other variables, as well as receive values from math operations or other functions. Check it out: 
variables
  yearOfBirth, currentYear, age
begin
  yearOfBirth = 2004
  currentYear = 2018
  age = currentYear - yearOfBirth
  println("The age of someone who was born in ", yearOfBirth, " is ", age, " years old")
end
What has happened? Please notice that we said that the age was equal to currentYear - yearOfBirth. In this example it means that age will be 2018 - 2004, which is 14. The program will write to the screen the following: "The age of someone who was born in 2004 is 14 years old".
Basic math operations
The four math operations are:
  • Addition: we'll use the + symbol
  • Subtraction: we'll use the - symbol
  • Multiplication: we'll use the * symbol
  • Division: we'll use the / symbol
So not let's try create a code where we can use some of the math operations. Let's use some examples from your math class: geometry!
variables
  perimeter, side, area
begin
  perimeter = 20
  side = perimeter / 4
  area = side * 4
  
  println("The side of a square with a perimeter of ", perimeter, " is ", side)
  println("The area of the square is ", area)
end
Can you think about what the computer will print to the screen? Think about it and then try it yourself.

Advanced
it is important to understand a concept call precendence of operators. This means that some operators are more important than others when the computer tries to do math. Multiplication and Divistion are more important than the Addition and Subtraction, so the computer will calculate those first. For example in the expression value = 10 - 3 * 2, the result will be 4. Why!? Because the computer will calculate 3 * 2 first, which is 6, and then calculate the subtraction, so it will do 10 - 6 and the variable value will receive the value 6. In order to inform the computer a different order of precedence, we should use parenthesis. In the previous example, let's do the math like this: value = (10 - 3) * 2, in this case the result will be 14 (instead of 4). This is because 10 - 3 will be calculated before, which is 7, and then the computer will proceed with the multiplication 7 * 2.
Exercises
  1. Create a program that displays your name and your age using 3 variables for your first name, last name and age.
  2. Create a program that has 3 variables called area, sideA and sideB to represent the area and sides of a rectangle, then set the values of the sideA and sideB and calculate the area. Use math to calculate the area.
  3. Create a program with the variables candies, children, candiesPerChild. Set the variable candies to 27 and the variable children to 3. Now make sure the program calculates into the variable candiesPerChild the number of candies that each child will receive if we equally distribute the candies.
