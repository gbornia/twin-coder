About
TwinCoder.org is a tool to teach kids how to code. There are many tools with that intent available such as Scratch that can be used for a similar purpose, and that could be a better option for your kid. The main goal with TwinCoder.org is to provide a development environment similar to those that professional programmers use, with a text editor, a compiler, error messages and the satisfaction of seeing your own program come to life. With this tool it is possible for kids to create awesome programs and cool projects perhaps to be displayed at school.

TwinCoder.org should not be used for commercial purposes, and it is only intended for educational purposes. It has no support and it will not have a good performance for complex processing.
