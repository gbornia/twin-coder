What is it?

TwinCoder.org is a free programming application built and designed to teach kids how to code.

It's a simple development platform and programming language that introduces basic concepts and allows kids to develop fun and impressive programs without the complexities involved in traditional programming languages.

How to install it?

1. Install Java SE (https://www.java.com/en/download/)
2. Download twin-coder.jar into your preferred folder (e.g. Desktop)
3. Double-click on the twin-coder.jar
