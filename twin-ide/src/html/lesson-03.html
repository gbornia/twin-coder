Lesson 03 - Use of read and speak
Hi, in this lesson we will learn how to use two important commands:
  • read: reads from the keyboard
  • speak: makes the computer speak a text
We will also use the commands we've learnt in past lessons.
Command read
In the previous lesson we learned how to use variables. We simply defined the variables and set a value for it so that we could use it. With the read command we will ask the person using the computer to enter the value and we will store it in a variable. Check the code below: 
variables
  firstName, lastName
begin
  print("What's your first name?")
  read(firstName)
  print("What's your last name?")
  read(lastName)

  println("Hi ", firstName, " ", lastName, " it is a pleasure to meet you!")
  println("Your last name ", lastName, " is very familiar, is your family from Westchester too?")
end
Notice that in the code above, we are asking the person that's using our program to enter her/his first name and last name. Later, we will print to the screen both first name and last name combined, and mention how familiar his last name is. Try it yourself!

We can use read to ask for number and afterwards do some calculations. Check it out: 
variables
  sideA, sideB, area
begin
  print("What's the first side of the rectangle?")
  read(sideA)
  print("What's the other side of the rectangle?")
  read(sideB)

  area = sideA * sideB
  
  println("The area of the rectangle is ", area)
end
The command read is one of the most important commands and you will use it in many occasions. Make sure you practice and you understand how it works.
Command speak
The command speak is not the most important command you need to learn, but it is really fun. It is a very simple one that we can use to make the computer speak! Check the code below: 

begin
  speak("Hello World!")
end
Now you are familiar with the "Hello World" classic example. You can also use speak just like print and println, passing multiple parameters. Like this:
variables
  age
begin
  age = 10
  speak("My age is ", age, " years old")
end
Try it yourself!

Important
  • Make sure you have your sound on, otherwise you will not hear the computer speak and it will look like nothing happened.
  • Be careful to not disturb other people next to you, you can use a headphone while you are practicing
  • Be respectful and do not attempt to make the computer say bad words, remember that others might be listening
Exercises
Now it is time to practice what we've learned. Below are some nice exercises!
  1. Create a program that asks how many pizzas the customer wants and prints to the screen the total (each of our pizzas is 14 dollars)
  2. Create a program that asks for the side of a square, and calculates the area and the perimeter
  3. Modify the programs above so that the computer speaks the total of the pizzas, the area and the perimeter of the square
Math tip
I know you are an ace in math and don't need help, but other fellow students might not remember and it is required for the exercises above, so here it goes:
  • The area of the square is the side times itself (area = side * side), i.e. the square of the side
  • The perimeter of the square is the side times 4 (area = 4 * side or area = side + side + side + side)
