Lesson 04 - Conditionals: use of if and comparisons
In this lesson we will learn two important aspects of conditionals:
  • command if: it allows the computer to behave differently depending on a condition
  • comparisons: the different types of comparisons that we can use
Command if
Now things are getting interesting! The conditional if is certainly one of the most important things in coding. It adds flavor to your code! In order to use it we will also need to understand the comparisons, which is next in this lesson.

The if commands needs a condition (for example, if the age is equals to 10) and then a set of instructions. Take a look at the code below:
variables
  age
begin
  print("What's your age (in years)? ")
  read(age)

  println("You age is ", age)

  if (age == 10)
    println("Nice! I also have the same age")
  end
end
In TwinCoder the comparison equals is written as == (and so it is with many other coding languages such as Java, C, Python. etc). It requires the two equals instead of one, to differentiate from the attribution (age=10 sets 10 to age, but age==10 compares the age to see if it is 10).

Another important concept is the else block. That block indicates what to do if the condition is not met. Take a look at the following example:
variables
  birthYear

Comparisons
There are multiple comparisons that you can use with the command if. Plese take a look at the table below:
Operator Description Example
== Compares if one side is equals to the other age==10, an age of 8, 9, 11 will be false, but 10 will be true
!= Compares if one side is different than the other age!=10, an age of 8, 9, 11 will be true, but 10 will be false
> Compares if one side is great than the other age>10, an age of 8, 9, 10 will be false, but 11, 12 will be true
>= Compares if one side is great or equals than the other age>=10, an age of 8, 9 will be false, but 10, 11, 12 will be true
< Compares if one side is less than the other age<10, an age of 8, 9 will be true, but 10, 11, 12 will be false
<= Compares if one side is less or equals than the other age<=10, an age of 8, 9, 10 will be true, but 11, 12 will be false
Now that we now the different types of comparison let's try the following:
variables
  siblings
begin
  print("How many siblings do you have? ")
  read(siblings)

  if (siblings == 3)
     println("I have 3 siblings too!")
  end

  if (siblings < 3)
     println("I have more siblings than you.")
  end

  if (siblings > 3)
     println("You have more siblings than me. Oh boy!")
  end
end
Try it yourself! Run the program above more than once and try it with a different number of siblings.
