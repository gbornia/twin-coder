Lesson 01 - Use of print, println and background
Hi, in this lesson we'll create our first program. And we will learn about 3 commands:
  • print: writes a text to the screen, the next text will write from where it stopped
  • println: writes a text to the screen and moves to the next line, the next text will write the line below
  • background: paints the whole screen with a color
Command print and println
All programs begin with begin and end with an end. Inside that block, we'll enter some commands. See the example below.
begin
  print("Hello World!")
end
The program above is a classic in programming. It tells the first words of the program! Now try the following using print:
begin
  print("Hi ")
  print("I'm ")
  print("coding.")
end
What has happened? Now try the following using println and see the difference:
begin
  println("Hi ")
  println("I'm ")
  println("coding.")
end
Did you see how the text was displayed? print makes the next command start from where it stopped, while println writes to the screen and moves on to the next line.

Both print and println accept multiple parameters, to understand that, let's try the following: 
begin
  print("Hi, my name is ", "Maria ")
  println("and I'm ", 10, " years old")
  println("Bye ")
  println("Bye ")
end
Please answer the following questions:
  1. Why the sentence Hi, my name is Maria and I'm 10 years old appeared in the same line?
  2. Why the sentence Bye Bye appeared in the two different lines?
  3. How can you make Bye Bye appear in the same line?
  4. Did you notice that 10 does not have quotes?
Command background
Now let's have some fun with colors. The command background paints the screen with any given color. Let's try the following:
begin
  background("yellow")
  println("Hello World!")
end
Did you see the what happened? Now try different colors. The colors accepted are:
  • black
  • blue
  • cyan
  • darkGray
  • gray
  • green
  • lightGray
  • magenta
  • orange
  • pink
  • red
  • white
  • yellow
There is another way to pick from millions of colors! Any color can be represented by a code, for example, red is #ff0000. To choose the correct code, go to Google and search for hexa color picker and find the code for your favorite color! My favorite color is #ffda9b, see how to do it: 
begin
  background("#ffda9b")
  println("Hello World!")
end
Exercises
  1. Create a program that tells about you: your name, how old are you and what's your school.
  2. Now do the same as above, but make sure you use more than one parameter for the, for example do not use quotations for the number that represents your age.
  3. Modify the program to use both print and println.
  4. Set your program with your favorite color (if you use dark colors it will be hard to read).
