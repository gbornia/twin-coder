String Functions

The following list of String functions are available to use in TwinCoder.org.

Function Description Code Snippet
concat Returns the concatenation of multiple strings(e.g. concat("Hello", " ","World") = "Hello World"). //What's the name?
firstName = "Scott"
lastName = "Adams"
fullName = concat(firstName, " ", lastName)
length Returns the length of a string(e.g. length("Hello World") = 11). //What's the name?
line = "Name: Scott Adams"
println("The name is ",substring(line, 6, length(line)))
lower Returns the lower case version of a string(e.g. lower("Hello World") = "hello world"). //Do you want to continue?
read(answer)
if (lower(answer) = "no")
  println("Goodbye")
  exit
end
position Returns the position of a substring inside a string (e.g. position("Hello World", "World") = 6). The function returns -1 if the substring is not found. //What's the name?
line = "Ridgeway School"
if position(line,"School")) > -1
  println("It's a school")
end
replace Replaces a search string within a string (e.g. replace("Hello World","ello","appy") = "Happy World").
Note: All occurrences will be replaced. Also as an advanced topic, the search string is a regex expression, which means that more special replacements can be made. It also indicates that using some special characters might lead to results different than expected (for example, trying to replace the $ by something else, needs special treatment since $ is a special regex indicator). 		//Remove spaces from a string
line = "Studies about math"
file = concat(replace(line, " ", "-"), ".txt)
substring Returns a substring of a string(e.g. substring("Hello World",6,11) = "World"). //What's the name?
line = "Name: Scott Adams"
println("The name is ",substring(line, 6, length(line)))
trim Returns the trimmed version of a string(e.g. trim(" Hello World ") = "Hello World"). //Remove spaces from the answer
read(answer)
answer = (trim(answer)
upper Returns the upper case version of a string(e.g. upper("Hello World") = "HELLO WORLD"). //Do you want to continue?
read(answer)
if (upper(answer) = "NO")
  println("Goodbye")
  exit
end
