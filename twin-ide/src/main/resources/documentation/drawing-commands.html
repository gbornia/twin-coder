Drawing Commands

The following list of Drawing commands are available to use in TwinCoder.org.

Command Description Code Snippet
background Sets the background color of the canvas. The parameter is the following
  1. color, the color to paint the background (default is white)
 //Paints the background black
background("black")
circle Draws a circle on the canvas, the parameters are as follows:
  1. x, the horizontal position
  2. y, the vertical position
  3. radius, the radius of the circle
  4. borderColor, the color of the border [Optional: default is black]
  5. fillColor, the color of the interior [Optional: default is white]
 //Draws the sun
circle(550, 350, 80, "red","yellow")
clear Clears the entire output canvas. //Draws a moving circle
speed = 10
start = 100
t = 0
while (t < 10)
  clear
  circle(start + t*speed, 200, 100)
  t = t + 1
end
image Draws an image from a file or a URL:
  1. image, the image path (local file or URL)
  2. x, the horizontal position
  3. y, the vertical position
  4. width, the width to draw the image, if empty uses the image width [Optional: default is the image width]
  5. height, the height to draw the image, if empty uses the image height [Optional: default is the image height]
 //Draws Einstein picture
image("https://twincoder.org/images/einstein.jpeg", 100, 80)
line Draws a line on the canvas, the parameters are as follows:
  1. x0, the horizontal position of the start point
  2. y0, the vertical position of the start point
  3. x1, the horizontal position of the end point
  4. y1, the vertical position of the end point
  5. lineColor, the color of the line [Optional: default is black]
 //Draws a blue line
line(20, 35, 120, 250, "blue")
oval Draws an oval shape on the canvas, the parameters are as follows:
  1. x, the horizontal position
  2. y, the vertical position
  3. width, the width of the oval
  4. height, the height of the oval
  5. borderColor, the color of the border [Optional: default is black]
  6. fillColor, the color of the interior [Optional: default is white]
Note that if width and height are the same, your are actually drawing a circle. The main difference is that the circle is drawn relative to it's center (x,y) while oval is relative to the top left corner of the rectangle that surrounds the oval shape. 		//Draws an oval
oval(550, 350, 80, 50, "red","yellow")
polygon Draws a polygon on the canvas, the parameters are as follows:
  1. x0, the horizontal position of the start point
  2. y0, the vertical position of the start point
  3. x1, the horizontal position of the 2nd point
  4. y1, the vertical position of the 2nd point
  5. x2, the horizontal position of the 3rd point
  6. y2, the vertical position of the 3rd point
  7. xN, the horizontal position of the Nth point [Optional and Repeatable]
  8. yN, the vertical position of the Nth point [Optional and Repeatable]
  9. lineColor, the color of the line [Optional: default is black]
  10. fillColor, the color of the interior [Optional: default is white]
The polygon needs at least 3 pairs of point (x,y) to be formed (triangle). To use more points, keep adding pairs of (x,y) to the list. For example, to draw a polygon with 5 vertices it would be something like polygon(10,10,15,30,40,28,120,35,50,19) where the (x,y) points (10,10), (13,30), (40,28), (120,35) and (50,19) form the vertices of the polygon. 		//Draws a blue star
polygon(300,100,350,300,550,350,350,400,300,600,250,400,50,350,250,300,"blue","blue")
rectangle Draws a rectangle on the canvas, the parameters are as follows:
  1. x0, the horizontal position of the top left corner
  2. y0, the vertical position of the top left corner
  3. x1, the horizontal position of the bottom right corner
  4. y1, the vertical position of the bottom right corner
  5. borderColor, the color of the border [Optional: default is black]
  6. fillColor, the color of the interior [Optional: default is white]
 //Draws a rectangle with a red line
rectangle(20, 35, 120, 250, "red")
image Draws an image from a file or a URL and rotates it to an angle:
  1. image, the image path (local file or URL)
  2. angle, the angle of rotation
  3. x, the horizontal position
  4. y, the vertical position
  5. width, the width to draw the image, if empty uses the image width [Optional: default is the image width]
  6. height, the height to draw the image, if empty uses the image height [Optional: default is the image height]
 //Draws Einstein picture with 45 degrees of rotation
rotateImage("https://twincoder.org/images/einstein.jpeg", 45, 100, 80)
text Writes text to the canvas, the parameters are as follows:
  1. x0, the horizontal position of the top left corner
  2. y0, the vertical position of the top left corner
  3. text, the text to be written
  4. color, the color of the text [Optional: default is black]
  5. fontName, the name of the font [Optional: default is Arial]
  6. fontSize, the size of the font [Optional: default is 12]
  7. fontStyle, the style of the font (plain, bold or italic) [Optional: default is plain]
 //Writes Hello World
text(200, 50, "Hello World", "red", "Verdana", 24, "italic")
