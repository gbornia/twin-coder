Date Functions

The following list of Date functions are available to use in TwinCoder.org.

Note:The dates are numbers that represent days. You can do math operations to add and subtract, for example now - 1 represents yesterday. Similarly, now - 1/24 represents one hour ago.
Function Description Code Snippet
formatDate Returns a string of a data following a specific format, e.g .formatDate(now) or formatDate(now, "MM/dd"). The formatDate accepts 1 or 2 parameters as follows:
  1. date: a date (number) to be formatted
  2. format (optional): the format to be applied, the default is "MM/dd/yyyy"
 //Show current hour and minutes
println(formatDate(now, "hh:mm"))
now Returns the current date and time println(now) //Show current hour and minutes
println(formatDate(now, "hh:mm"))
toDate Returns a date from a string, it allows to specify a format as well, e.g .toDate("07/04/1776") or toDate("1776-07-04", "yyyy-MM-dd"). The toDate accepts 1 or 2 parameters as follows:
  1. string: the string to be transformed into a date (number)
  2. format (optional): the format to be applied, the default is "MM/dd/yyyy"
 //Calculate your age
print("What your date of birth (e.g. 02/14/2008)? ") read(dateOfBirth) println(Your age is ", round( (now - toDate(dateOfBirth)) / 365.25 ), " years old")
