Console Commands

The following list of Console commands are available to use in TwinCoder.org.

Command Description Code Snippet
clear Clears the entire console (this command also cleans any canvas object, see clear in drawing commands. //Ask for values and present results
print("Value of A = ")
read(a)
print("Value of B = ")
read(b)
clear
println("Result of A + B = ", a + b)
keyPressed Returns the last keyPressed value on the keyboard. After reading it, the keyPressed will no longer exist. //While nothing is pressed, keep doing things
while (keyPressed = "")
  println("New line")
end
log Writes to the log of the program. It accepts many arguments. It is used to keep track of what is happening with the pogram. //Calculate the squared root of a number
print("Value of A = ")
read(a)
if (isNumber(a))
  println("The squared root of a is ", sqrt(a))
else
  log("The value of A: ", a, " is not a number")
end
print Writes to the console. It accepts many arguments. The command print writes to the console and keeps the cursor on the last character (next writing activity will happen on the same line). See println below. //Writes the first and last name one next to the other
firstName = "John"
lastName = "Secada"
print(firstName)
print(" ")
print(lastName)
//John Secada
println Writes to the console. It accepts many arguments. The command println writes to the console and moves the cursor to the next line. //Writes the first and last name one next to the other
firstName = "John"
lastName = "Secada"
println(firstName)
println(" ")
println(lastName)
//John
//
//Secada
println("Hi ", firstName, " ", lastName, ", it's a pleasure to meet you!")
//Hi John Secada, it's a pleasure to meet you!
read Reads a value entered by the user and stores it into a variable. //Reads first and last name
print("What's your first name?")
read(firstName)
print("What's your last name?")
read(lastName)
println("Hi ", firstName, " ",, lastName, ", it's a pleasure to meet you!")
