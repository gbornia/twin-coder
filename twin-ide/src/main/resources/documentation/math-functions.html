Math Functions

The following list of Math functions are available to use in TwinCoder.org.

Function Description Code Snippet
abs Returns the absolute value of number (e.g. abs(-12) = 12), i.e. ignore the signal. //What's the distance between A and B?
a = 8
b = 5
println("From A to B there are ",abs(b-a))," feet")
avg Returns the average value of a series (e.g. avg(3,3,8,2) = 4). //What's the average score of the class?
mary = 8.5
john = 7
tiffany = 9.5
println("The average score is ",avg(mary,john,tiffany))
cos Returns the cosine of an angle. Uses degrees (e.g. cos(45) = 0.5). //Calculate the horizontal position of a inclined line
angle = 45
width = 200
x = width * cos(angle)
div Returns the integer division of two numbers (e.g. div(5,2) = 2). See also the function mod. It's important to observe that the exact same result could be done with round (e.g. round(5/2) = 2). //How manu weeks
days = 15
weeks = div(days, 7)
max Returns the maximum value of a series (e.g. min(2,4,1,3) = 4). //How old is the oldest kid?
john = 12
kathy = 11
println("The oldesr kid is ", max(john, kathy), " years old")
min Returns the minimum value of a series (e.g. min(2,4,1,3) = 1). //How old is the youngest kid?
john = 12
kathy = 11
println("The youngest kid is ", min(john, kathy), " years old")
mod Returns the modulus or remainder of an integer division of two numbers (e.g. mod(5,2) = 1). See also the function div. //How old is your child?
months = 27
println("Your child is ", div(months,12), " years and ", mod(months,12), " months")
pi Returns the value of Pi (e.g. pi = 3.1415926535...). //What is the perimeter of the circle?
radius = 10
perimeter = 2 * pi * radius
power Returns the value of a number power to another (e.g. power(3,2) = 9). //What is the volume of the cube?
side = 3
volume = power(side, 3)
random Returns a random number between zero and the given number (e.g. random(10) will return some number between 0 and 10).
Note: random returns a real number, you might want to use it in combination with round to get an integer number. 		//Give me a number from 0-10?
println("Let me guess from (0-10) the number... ", round(random(10)))
round Rounds a number to an integer (e.g. round(7.75) = 8). //How many years?
months = 38
years = round(months / 12)
sin Returns the sine of an angle. Uses degrees (e.g. sin(45) = 0.5). //Calculate the vertical position of a inclined line
angle = 45
width = 200
x = width * sin(angle)
sqrt Returns the square root of a number (e.g. sqrt(9) = 3). //Calculate the side of a square
area = 100
side = sqrt(area)
