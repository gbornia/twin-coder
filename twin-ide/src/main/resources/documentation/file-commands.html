File Commands

The following list of File commands are available to use in TwinCoder.org.

Command Description Code Snippet
readFile Reads the content of a text file into a variable. It requires 2 arguments:
  1. variable, the variable that will receive the file content
  2. file, the file path or URL for the file
If the file path is a URL, the file will be downloaded into the cache. It will be kept if TwinCoder is configured to keep cache, otherwise it will be deleted from cache after usage. 		//Reads a simple text file and prints it
readFile(a, "/users/TwinCoder/samples/hello.txt")
println(a)
